# SuPartido — Plataforma de Reserva de Canchas de Fútbol

Plataforma moderna para la reserva de canchas sintéticas en Iquique y Alto Hospicio.

## Características
- 🚀 **Next.js 15** (App Router) + TypeScript
- 🎨 **Tailwind CSS v4** + shadcn/ui
- 🛠️ **Turborepo** Monorepo
- 🏗️ **NestJS** Backend (Sprint 2+)
- ⚡ **Supabase** (Auth & DB)

## Estructura del Proyecto
- `apps/web`: Aplicación frontend principal.
- `apps/api`: Backend API en NestJS.
- `packages/shared`: Tipos e interfaces compartidas.

## Comenzar
1. Instalar dependencias:
   \`\`\`bash
   npm install
   \`\`\`

2. Correr en modo desarrollo:
   \`\`\`bash
   npm run dev
   \`\`\`

3. Construir para producción:
   \`\`\`bash
   npm run build
   \`\`\`
